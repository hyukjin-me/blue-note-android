# BlueNote AnswerSheet

Blue Note is an android app that provides simple notes and answer .

Tech & Libaries 
- Min SDK version 21 
- Jetpack : Navigation, LiveData, Room
- Retrofit2
- Moshi
- MPAndroidChart
- CircleImageView
- Firebase : analytics, adMob

Open API
- OpenWeather https://openweathermap.org/
